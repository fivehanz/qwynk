# Qwynk Brand Book

This is a SvelteKit application that implements the Qwynk Brand Identity Specification (Abyssal Terminal theme).

## Features

- **Typography**: Tri-Type Stack (Space Grotesk, IBM Plex Sans, IBM Plex Mono).
- **Color Palette**: Bioluminescence (Dark Mode Only).
- **Components**: Neo-Brutalist buttons and data cards.
- **Motion**: Hydraulic motion doctrine implementation.

## Setup & Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the URL shown (usually `http://localhost:5173`).

## Tech Stack

- **Framework**: SvelteKit
- **Styling**: Tailwind CSS v4
- **Theme**: Custom "Abyssal Terminal" implementation
